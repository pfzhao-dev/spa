Instructions on how to build:

1. Open folder in Visual Studio 2019 and Ctrl+Shift+B (to build the SPA and autotester)
2. On the top of the screen click the green play button for autotester.exe (remember to choose autotester.exe)

autotester.exe's path is {build_folder}/src/autotester folder.

Use Visual Studio 2019 - Debug version
